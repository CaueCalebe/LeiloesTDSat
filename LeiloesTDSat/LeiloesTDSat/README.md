# LeiloesTDSat
 
## Sobre o Projeto
Sistema de gerenciamento de leilões desenvolvido para controle de produtos,
permitindo cadastrar itens, definir valores e acompanhar o status de venda
(À Venda / Vendido). O sistema é conectado a um banco de dados MySQL e
desenvolvido em Java com interface gráfica via NetBeans IDE.
 
## Tecnologias Utilizadas
- Java
- MySQL
- NetBeans IDE
